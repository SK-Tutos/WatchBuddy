    `npx expo install @react-navigation/bottom-tabs react-native-safe-area-context react-native-screens`
    `npx expo install @react-navigation/drawer react-native-gesture-handler react-native-reanimated`



Note : Utiliser l'application Expo Go en scannant le QRCode généré lors d'un `expo start` (ou dans le cas ce cours : `yarn ios`) pour lancer l'application sur n'importe quel appareil (iOS et Android).
